Cybersecurity Internship Tasks (Task 01 – Task 05)
1.Overview

This repository contains all the tasks completed as part of my Cybersecurity Internship.
Each task focuses on a fundamental cybersecurity concept and is implemented using Python for educational and ethical learning purposes.
The tasks progress from basic security concepts to practical network analysis, demonstrating hands-on experience in cybersecurity.

Task 01: Password Strength Checker

I Description:
A Python program that evaluates the strength of a password based on multiple security parameters.

II Features:
1.Checks password length
2.Validates use of:
  2.1 Uppercase letters
  2.2 Lowercase letters
  2.3 Numbers
  2.4 Special characters
3.Classifies passwords as Weak, Medium, or Strong

III Skills Gained

1.Password security principles
2.Input validation
3.Python programming fundamentals

Task 02: Caesar Cipher Tool

I Description
A classical cryptography tool that performs encryption and decryption using the Caesar Cipher technique.

II Features

1.Encrypts plaintext using a shift key
2.Decrypts ciphertext back to plaintext
3.Supports alphabetic character handling

III Skills Gained

1.Cryptography basics
2.Encryption and decryption logic
3.Understanding limitations of classical ciphers

Task 03: Password List Analyzer

I Description
A program that analyzes a list of commonly used passwords to identify weak password patterns.

II Features

1.Reads passwords from a text file
2.Detects weak and common passwords
3.Analyzes password length and complexity

III Skills Gained

1.File handling in Python
2.Password attack awareness
3.Security analysis

Task 04: Simple Keylogger (Educational Use Only)

I Description

A basic keylogger created strictly for educational and ethical purposes to understand how keystroke logging works.

II Features

1.Captures keyboard inputs
2.Stores keystrokes in a log file
3.Includes ethical warnings and usage restrictions

III Skills Gained

1.System interaction concepts
2.Ethical considerations in cybersecurity
3.Secure coding responsibility

Task 05: Network Packet Analyzer

I Description

A network packet sniffer that captures and analyzes live network traffic using Python and Scapy.

II Features

1.Displays network interface name
2.Captures live packets
3.Shows:
  3.1 Source IP and Destination IP
  3.2 Protocol (TCP / UDP / ICMP)
  3.3 Source and Destination Ports
  3.4 Payload size
4.Protocol filtering support
5.ICMP detection (Ping monitoring)
6.Suspicious packet alert (safe threshold)
7.Saves packet logs to TXT and CSV files

III Skills Gained

1.Networking fundamentals
2.Packet analysis
3.Scapy library usage
4.Ethical network monitoring

2.Technologies Used:
1.Python 3
2.Scapy
3.File Handling
4.Networking Concepts
5.Cybersecurity Fundamentals
