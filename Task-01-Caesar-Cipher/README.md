##  About Caesar Cipher

Caesar Cipher is one of the oldest encryption techniques.
Each character in the plaintext is shifted by a fixed number of positions in the alphabet.

### Example

```
Plain Text : HELLO
Shift      : 3
Cipher Text: KHOOR
```
